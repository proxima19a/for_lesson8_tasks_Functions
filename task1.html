Функции в JavaScript

Задание 1.

Необходимо создать 3 функции через Function declaration, Function expression, Arrow function expression. Они должны принимать в себя параметр name и возвращать строку, подставляя туда этот параметр. Название у них должны быть следующими: getName1, getName2, getName3.

Функции должны удовлетворять следующим условиям:

  1. В каждую из этих функций должен быть передан 1 параметр name.
  2. Из каждой функции с помощью return должно быть возвращено значение “Имя равно name”, где name - значение переданного в функцию параметра name.

После вам необходимо вызвать каждую функцию и вывести возвращаемое из нее значение в консоль.
